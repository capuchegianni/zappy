# Zappy GUI

## Description

This is a GUI for the Zappy project. It is written in C++ using SFML.

## Build

### Dependencies

- SFML 2.5
- CMake 3.27
- Make
- C++20

### Build

```bash
mkdir build
cd build
cmake ..
make
```

**Or**

```bash
make
```

## Usage

```bash
./zappy_gui -h <IP> -p <PORT>
```
